PS : 使用该脚本前应安装TamperMonkey或其他脚本插件

使用方法:
	按住Alt键,左键拖动并选择百度网盘分享的代码,例如 : 12345678 密码:1234
	不能选择前面的htttp://pan.baidu.com/s

实现逻辑:
	利用正则表达式去除无用字符,如上面的"密码:",其他无用字符还包括任意非 数字,字母,下划线和短横线
	逻辑判断是否有提取码
	生成网址并跳转,网址格式为 : http://pan.baidu.com/s/代码#提取码
	在分享页面中获取"#提取码"自动填提取码并跳转
